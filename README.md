## ✨ 데스크톱 핵심 기능

- **AI 추천 시각 표시**: P0/P1/P2 우선순위 뱃지, 추천 행 컬러 보더, AI 태그
- **행 단위 액션**: 신청 / 보류 버튼
- **보류 처리 모달**: 6종 사유(다시 알림 / 거래 중단 / 비활성 / 누락·Data 오류 / 수기 접수 / 거래 종료)
- **bulk 액션**: 선택 일괄 신청 / 선택 일괄 보류 / 추천 모두 선택

## 📱 모바일 핵심 기능

별도 페이지로 구현 (반응형 X). `SK가스_type_B_07/08`, `모바일 주문.png` 디자인 기반.

- **홈**: 13개 아이콘 그리드 + 사용자 인사 카드 + 출발지/도착지
- **LPG주문 신청**: 추천 주문 카드 리스트 (P0/P1/P2 컬러 보더, AI 태그, 우선순위 뱃지)
- **카드 단위 액션**: 신청 / 보류 버튼
- **바텀시트 보류 모달**: 6종 사유 카드 라디오, 적용 기간, 메모
- **탭 필터**: 전체 / 추천 / 보류
- **하단 고정 액션바**: LPG주문신청 / LPG주문현황 큰 버튼 두 개

## 🛠 로컬 실행

별도 빌드 없이 `tab_layout.html`을 브라우저로 열면 바로 동작합니다.
