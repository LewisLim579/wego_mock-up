# WEGO 2.0 · 주문자동화 화면설계 (Mockup)

SK가스 충전소용 AI 주문 추천 / 보류 / 주문신청 프로세스 mockup 입니다.

## 🌐 라이브 데모

GitHub Pages: <https://lewislim579.github.io/wego_mock-up/>

## 📁 구조

- `tab_layout.html` — 메인 화면 (B/L 주문신청 + B/L 주문현황)
- `style.css` — 디자인 시스템 (SK가스 type-B 기반)
- `index.html` — `tab_layout.html`로 자동 리다이렉트
- `디자인이미지/` — 디자인 참고 이미지

## ✨ 주요 기능

- **AI 추천 시각 표시**: P0/P1/P2 우선순위 뱃지, 추천 행 컬러 보더, AI 태그
- **행 단위 액션**: 신청 / 보류 버튼
- **보류 처리 모달**: 6종 사유(다시 알림 / 거래 중단 / 비활성 / 누락·Data 오류 / 수기 접수 / 거래 종료)
- **bulk 액션**: 선택 일괄 신청 / 선택 일괄 보류 / 추천 모두 선택
- **반응형**: 데스크톱 / 태블릿 / 모바일 대응

## 🛠 로컬 실행

별도 빌드 없이 `tab_layout.html`을 브라우저로 열면 바로 동작합니다.
